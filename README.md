# BomberFPV

Базовый прототип игры (вид сверху): управляем FPV-дроном и сбрасываем боеприпас, пролетая над целями. Сброс требует тайминга (боеприпас падает с задержкой), после сброса есть перезарядка. За очки можно апгрейдить дрон.

## Запуск

```bash
python3 -m pip install -r requirements.txt
python3 src/game.py
```

Если нужно проверить, что всё запускается в headless среде (без окна):

```bash
python3 src/game.py --smoke --seconds 1.0
```

## Управление

- **WASD / стрелки** — полёт
- **Space** — сброс боеприпаса (если перезарядка закончилась)
- **U** — меню апгрейдов
- **1/2/3** — купить апгрейд в меню
- **Esc** — выход / закрыть меню
